A lightweight, local Small Language Model (SLM) inference tool for testing prompts and generating structured outputs. Designed for offline experimentation and rapid AI prototyping without external dependencies.

Features

Run real-time inference on your local machine.

Supports structured JSON outputs for easy downstream analysis.

Lightweight: minimal setup, no heavy cloud dependencies.

Flexible prompt testing with testprompts.json.

Benchmarks optional: store performance data in CSV or JSON.

Fully Python-native, ready to integrate with other AI tools or pipelines.
